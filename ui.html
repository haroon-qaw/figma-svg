Copy as SVG with 'currentColor'

In an SVG, 'currentColor' makes fill or stroke inherit the element's CSS color value.

What does this mean?

Masks often use black, white, or grayscale shapes to control visibility.

ClipPaths define visible areas but can still contain styled shapes.

If you convert those colors to currentColor, you might unintentionally change how the masking behaves.
